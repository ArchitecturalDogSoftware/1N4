# Code of Conduct

1N4 follows the Rust Foundation's Code of conduct,
available at: <https://www.rust-lang.org/policies/code-of-conduct>.
Put simply: treat people with respect.

## Reporting Violations

If you feel that the code of conduct has been violated, do not hesitate to reach out.
Publicly loop us into a discussion using an @mention, or [reach out to us in private](#contacting-us).

## Contacting Us

As of right now, 1N4 has no official method to contact maintainers privately.
We are working on a solution.
